Establece 1046 como un manejador para crear una traza de la ejecución. Si es 1047 desactivamos la traza.

El 1048 toma 6 parámetros:

  • Un nombre de evento,
  • Un nombre de fichero,
  • Un número de línea
  • Un ID de objeto
  • Un binding, y
  • Un nombre de una clase
1049 es invocado cada vez que ocurre un evento. Los eventos son:
  • c-call
  • c-return
  • call
  • class
  • end
  • line
  • raise
  • return
La traza se desactiva en el contexto de 1050.
