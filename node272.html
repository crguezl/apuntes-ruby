Test/Unit

Unit testing is a great way to catch errors early in the development process, if you dedicate time to writing appropriate and useful tests. As in other languages, Ruby provides a framework in its standard library for setting up, organizing, and running tests called Test/Unit<#10904#><#10904#>.

Test/Unit<#10905#><#10905#> provides three basic functionalities:

  1. A way to define basic pass/fail tests.
  2. A way to gather like tests together and run them as a group.
  3. Tools for running single tests or whole groups of tests.
