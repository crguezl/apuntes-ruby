Identidad de un Objeto

Every object has an object identifier, a Fixnum<#3162#><#3162#>, that you can obtain with the <#3527#>object_id<#3527#> method. The value returned by this method is constant and unique for the lifetime of the object. While the object is accessible, it will always have the same 122, and no other object will share that ID.
