Práctica: HTML DSL

En esta práctica deberá escribir una clase que provea una funcionalidad parecida (por supuesto, no completa) a la de la librería markaby. La librería provee métodos 1160, 1161, 1162, etc. que permiten la escritura de los correspondientes tags HTML: 
verbatim1133#
Los métodos, como 1163 reciben como primer argumento una cadena que es la que hay que rodear entre las marcas 1164 y 1165. También pueden recibir opcionalmente un segundo argumento que sea un hash especificando los atributos asociados con el tag: 
verbatim1134#
que debería dar lugar a un texto parecido a este: 
verbatim1135#
Si en la llamada no se provee la cadena inicial, se deberá proveer un bloque. Dicho bloque construye la cadena que se interpolará entre las marcas. Así: 
verbatim1136#

debería producir la cadena: 

verbatim1137#
La ejecución de este código 
verbatim1138#
debería producir una salida parecida a esta: 
verbatim1139#

Repase las secciones #seubsection:methodmissing#9391> y #subsection:instanceeval#9392>.
