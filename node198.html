Práctica: Conjuntos

Defina una clase Conjunto que disponga de métodos para:

  • Calcular el cardinal de un conjunto
  • Determinar las relaciones pertenencia de un elemento y subconjunto
  • La operación de unión:
  • La operación de intersección
  • Diferencia de conjuntos
  • Convertir un conjunto en una cadena
Debe funcionar de manera similar a la clase Set proveída con Ruby:

Puede usar 796s para implementar los conjuntos. Asuma que los elementos del conjunto universal admiten un orden total. Esta sesión con 797 sugiere una posible implementación:

Esto es, puede guardar el conjunto como un 798 ordenado en el que se han suprimido los elementos repetidos.

La relación de orden entre conjuntos es parcial: dados dos conjuntos 799 y 800 no necesariamente ocurre que 801 o aque 802.
