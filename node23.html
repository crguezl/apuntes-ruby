Ejercicios: Expresiones Regulares. Un programa que convierte de Celsius a Farenheit

Ejercicio 7.6.1   El siguiente programa convierte de Celsius a Farenheit y viceversa: 
MacBookdeCasiano:rubytesting casiano$ ruby -d f2c.rb 32F 
32F
Matches num = 32 kind= F
celsius = 0.0 farenheit = 32.0


MacBookdeCasiano:rubytesting casiano$ ruby f2c.rb 32F 
celsius = 0.0 farenheit = 32.0
Ejecútelo con el debugger e intente averiguar el significado de cada una de las sentencias. 

$ cat -n f2c.rb 
     1  #!/usr/bin/env ruby
     2  
     3  t = ARGV[0] || '0C'
     4  puts t if $DEBUG
     5  
     6  if t =~ /^\s*(\d+(?:\.?\d+)?)([CF])\s*$/i
     7    puts "Matches num = #{$1} kind= #{$2}" if $DEBUG
     8    num, type = $1.to_f, $2
     9  
    10    if (type == 'C') 
    11      celsius = num
    12      farenheit = (celsius*9/5)+32
    13    else
    14      farenheit = num
    15      celsius = (farenheit-32)*5/9
    16    end
    17    puts "celsius = #{celsius} farenheit = #{farenheit}"
    18  else
    19    puts "does not match" if $DEBUG
    20  end
